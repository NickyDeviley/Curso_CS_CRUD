Meu Primeiro Programa em C# (CRUD) – Desenvolvimento de uma lista de contatos com interface gráfica.

Link: https://www.udemy.com/share/107RcS3@cHpHZvmoXPnU10nP6oStaIH1ZugUN1As5-qHHiKvxz2CFNIeqhFC7Vqnm-3tu1M_SA==/ 

Descrição: Este repositório contém o projeto desenvolvido durante o curso prático da Udemy, focado na criação de um software funcional de lista de contatos. O projeto demonstra a aplicação de operações CRUD (Create, Read, Update, Delete) utilizando a linguagem C# e as ferramentas visuais do Microsoft Visual Studio. O objetivo central foi integrar uma interface gráfica amigável com um banco de dados local embutido na própria IDE para persistência de dados.

Pré-requisitos:

    Microsoft Visual Studio (versão Community ou superior).

    .NET SDK (compatível com a versão utilizada no curso).

    Workload de "Desenvolvimento para desktop com .NET" instalado no Visual Studio.

Instalação/Uso:

    Clone o repositório ou baixe os arquivos do projeto.

    Localize o arquivo de solução (.sln) e abra-o com o Visual Studio.

    No Gerenciador de Soluções, verifique se o arquivo do banco de dados local está presente na estrutura do projeto.

    Pressione F5 ou o botão Iniciar para compilar e rodar a aplicação.

    Utilize a interface para adicionar, visualizar, editar ou excluir contatos da sua lista.

Tecnologias:

    C#

    Windows Forms / WPF (Interface Gráfica)

    Banco de Dados SQL Local (LocalDB / Service-based Database)

    Visual Studio

Contribuição:
Este é um repositório de nível introdutório para fins de registro de aprendizado. Feedbacks sobre a estrutura da interface ou melhorias na lógica de persistência de dados são bem-vindos através de pull requests.
